

## Instalação do Google Tag Manager (GTM)

Para instalar o GTM no seu projeto, precisamos adicionar o script do GTM no `index.html`. O processo é simples:

### O que você precisa fazer primeiro

1. Acesse [tagmanager.google.com](https://tagmanager.google.com)
2. Crie uma conta e um container para seu site (tipo: Web)
3. Copie o **ID do container** (formato: `GTM-XXXXXXX`)

### O que vou implementar

1. **Adicionar o script GTM no `<head>`** do `index.html` (snippet principal)
2. **Adicionar o `<noscript>` no `<body>`** do `index.html` (fallback)

Ambos os snippets são fornecidos pelo próprio Google ao criar o container.

### Você já tem o ID do GTM?

Se sim, me passe o ID (ex: `GTM-XXXXXXX`) e eu faço a instalação. Se ainda não, crie o container primeiro no Google Tag Manager e depois me envie o ID.
